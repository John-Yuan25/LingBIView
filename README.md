# LingBIView 

（持续更新中......）

#### 介绍

物联网低代码可视化平台

LingBIView 是一个高效的拖拽式低代码数据可视化开发平台，将图表或页面元素封装为基础组件，无需编写代码即可制作物联网数据大屏。
